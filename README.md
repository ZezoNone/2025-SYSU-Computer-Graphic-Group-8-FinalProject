# **OpenGL实践**

## 用OpenGL搭建并渲染一个场景

2025中山大学计算机图形学第八小组期末项目：用OpenGL搭建并渲染一个场景  

此文件用于说明项目内容，介绍如何实现，实现过程遇到的问题和解决过程，每个小组成员完成的内容等。  

在这个选题中，我们使用OpenGL搭建并渲染了一个微型图书馆场景，为了实现这一场景，我们实现了以下功能：  

- **模型文件加载：** 使用Assismp库加载gltf文件
- **PBR光照模型：** 一种基于物理的光照模型
- **交互摄像机：** 通过鼠标的拖动实现镜头的旋转，鼠标滚轮实现拉近效果，WASD控制镜头的移动
- **实现阴影效果：** 使用Shadow Map方法为物体构建阴影，实现了点光源阴影
- **实现将HDR立方体贴图作为天空盒** 将源HDR图像转换为立方体贴图纹理
- **实现基于图像的光照(IBL)：** 让光照在环境贴图下是物理正确的
- **实现视差遮蔽映射（POM）：** 可以使用视差贴图使物体更真实
